Personal Access Token の取得方法

  1. GitHub にログインした状態で こちら を開く
  2. Generate new token をクリック
  3. Token name に任意の名前（例：「pechunia-admin」）
  4. Expiration（有効期限）を設定（90日〜1年がおすすめ）
  5. Repository access：Only select repositories → 該当リポジトリを選択
  6. Repository permissions の Contents を Read and write に
  7. 下部の Generate token をクリック
  8. 表示された github_pat_xxxxx... をコピー（一度しか表示されません）

※ トークンはお使いのブラウザにのみ保存され、第三者に送信されることはありません。
